#springboot 脚手架
#base-code-common: 基础代码包
    bo：逻辑中转实体类
    dao：数据层接口类
    domain：数据库实体映射类
    utils：工具包类
#base-code-service: 业务逻辑包
    auth：登录和权限管理相关
    common：通用业务接口
    impl：接口实现类
        auth：登录和权限管理相关接口实现类
        common：通过接口实现类
#base-code-controller: 业务控制层
    config：配置类
    controller：控制层类
        auth：登录和权限控制层类
    datasource：数据源配置
#base-code-job: 定时任务
